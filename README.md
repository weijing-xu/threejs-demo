<h3>说明：有网络获取的three.js示例，也有自己写的， 喜欢的点个star，大家相互学习，互相进步</h3>
 


## 前端运行

```bash
# 克隆项目
git clone https://github.com/weijing-xu/threejs-demo.git

# 进入项目目录
cd threejs-demo

# 安装依赖
npm install

# 启动服务
npm run dev
 
# 前端访问地址 http://localhost:3000
```

## base 目录

1.  index.html：主页
2.  1.html：第一个three.js页面。一个旋转的正方体
3.  2.html：判断浏览器是否支持webgl
